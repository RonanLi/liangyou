当前位置： 义务贷（浙江得龙进出口有限公司） >帮助> 用户投标
第一步：
用户点击我要理财，进入借款投标页面。

 



第二步：
根据用户需求选择要投资的借款标，进入该标的投标页面。用户可以搜索需要的借款标，计算投资收益。

 


第三步：
用户可以查看详细的借款标信息：借款金额、借款期限、借款利率、还款方式、借款标的投标情况，借款用户信息、该标审核资料。确认后可以进行投标，进入该标的投标页面。

 



第四步：
输入投资金额和支付密码，投资金额小于用户可用余额。第一次投标要设置支付密码，初始支付密码与用户登录密码相同。

 


第五步：
投标完成后用户资金冻结，等该标满标复审通过后用户将冻结的投资资金转入借款标用户并开始计算利息。到还款日期，用户会收到借款本金和利息，并扣除10%的利息作为网站管理费。用户可在我的账号中点击收款中的项目进行查看。

 


第六步：
用户添加设置自动投标信息，自动投标功能可以最大化利用用户剩余资金，增加收益。
